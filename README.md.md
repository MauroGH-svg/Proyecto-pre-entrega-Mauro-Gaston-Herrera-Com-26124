# ------------- ESTELARES EVENTOS --------------------------

Proyecto Final - Talento Tech Front End

##  Descripción

Estelares Eventos es una página web desarrollada para la contratación de servicios para eventos sociales.

El usuario puede visualizar los distintos servicios disponibles, agregarlos a un carrito de compras, calcular automáticamente 
el total, conservar la información mediante LocalStorage y finalizar la contratación mediante una interfaz amigable con SweetAlert2.

---

## Funcionalidades

✔ Consumo de una API mediante Fetch.

✔ Generación dinámica de servicios.

✔ Carrito de compras.

✔ Cálculo automático del total.

✔ Persistencia de datos con LocalStorage.

✔ Vaciar carrito.

✔ Confirmación de compra mediante SweetAlert2.

✔ Diseño responsive.
